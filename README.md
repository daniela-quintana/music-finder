# DESAFÍO TÉCNICO

**Music Finder** es una aplicación que te permite buscar canciones por artista o banda, escucharlas y marcarlas como favoritas. 🚀

## LEVANTAR BACK

`cd backend` 
`npm install` 
`node index.js` 


## LEVANTAR FRONT

`cd frontend` 
`npm install` 
`npm run dev ` 

El frontend estará disponible en: **http://localhost:3000**
